This is a game I’m personally creating. The story deals with death, corruption, and heroism of knights that have to save the kingdom and the realm from the evil in the world and the evil from within. I programmed the player movement, path finder of the Artificial Intelligence (AI), User Experience (UX), and User Interface (UI) of the pause and help menu I created. This is still a work in progress but more video demos will be added in the future.

                {
                    if (transform.position != target[current].position)
                    {
                        Vector3 pos = Vector3.MoveTowards(transform.position, target[current].position, speed * Time.deltaTime);
                        GetComponent().MovePosition(pos);
                        Vector3 relativePos = target[current].position - transform.position;

                        // the second argument, upwards, defaults to Vector3.up
                        Quaternion rotation = Quaternion.LookRotation(relativePos, Vector3.up);
                        transform.rotation = rotation;
                    }
                    else
                    {
                        current = (current + 1) % target.Length;
                    }
                }

Avalon Forest is a forest where bandits and monsters hide, there will be a dungeon that the player must go to this dungeon and defeat the boss inside. I created this scene by creating a terrain, from there I used the terrain tool to add trees, grass, and rocks. I added a lake and found a strong wind blowing sound to accompany the wind effect. The last thing I did was made a script for the camera switching to show different parts of the scene from different viewpoints.

In the castle town, the player will be able to buy items, weapons, and magic spells that they can use on their journey. Players will be able to talk to some people in town for side quests and to learn more about the main objective. This is an overview version of the level I created for the demo. I created lights so the town and castle would be visible. I added town sound effect for the background.

The Meglorian Meadow is a depressing meadow where many battles were fought and many great warriors died in war, a black magic keeps the land from ever becoming dry with never ending rain, it’s up to players to find the dungeon and discover what’s causing the rain. For this level I added a rain effect along with fog to give a depressing vibe along with adding rain sound effect.

These snowy mountains tower across the sky giving a shadow that stretches the land. Legends say that dragons fly above these great mountains. The player will be able to travel in this mountainous region and eventually fight a dragon. I designed this section with snowy mountain models, a sky box, and a blizzard effect. I plan to add more to this scene later.
